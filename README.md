# YRCode react portfolio 

[live demo](https://yrcode.netlify.app) -yrcode portfolio

## requirements

nodejs >= v18.13.0

## install portfolio

```
git clone https://github.com/yassinerahhaui/yassine-portfolio
cd yassine-portfolio
npm install
```

### run portfolio
```
npm run dev
```
### build portfolio
```
npm run build
```
